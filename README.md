# Задание 1. Рекурсия

## Задача 1

### Текст задачи

Составить программу, которая находит значение отношения трехзначного
числа к сумме его цифр. 

### Алгоритм решения

1) Запрашиваем с пользователя трёхзначное число.
2) Рекурсивно считаем сумму цифр числа.
3) Опрделяем результат - делим изначально число на получившуюся сумму.
4) Выводим результат.

# Задание 3. Решить задачу на обработку списка

## Задача 1

### Текст задачи

Определим множество как список без повторяющихся элементов. Доказать II
дистрибутивный закон.

### Алгоритм решения

1) Запрашиваем с пользователя количество элементов первого множества.
2) Рекурсивно запрашиваем элементы первого множества и заносим в список.
3) То же самое с остальными двумя множествами.
4) Вычисляем левую часть закона (A ∩ (B ∪ C)).
5) Вычисляем правую часть закона ((AB ∪ AC)).
6) Сравниваем части.
7) Выводим результат.

# Задание 4. Решить логическую задачу

## Задача 1

### Текст задачи

Имеется четыре котенка – Дружок, Елисей, Фантик и Мурлыка и четыре
мальчика – Миша, Максим, Леня и Дима. Каждый мальчик взял себе котенка
любимого цвета.
При этом:
– Фантик – не рыжий
– Мурлыка – не серый
– Дружок – не белый
– Елисей – не серый
– У Миши – черный котенок
– У Максима – Мурлыка
– У Лени – Елисей
– У Димы – белый котенок
– Дима не взял Фантика
– Дружок – не серый
– Одно из этих пяти утверждений ложное
У какого мальчика какой котенок?

### Алгоритм решения

1) Создаём списки котят и цветов.
2) Перебираем все варианты котят для мальчиков.
3) Перебираем все варианты цветов для котят.
4) Определяем условия.
5) Проверяем условия про цвета котят.
6) Выводим результат.
